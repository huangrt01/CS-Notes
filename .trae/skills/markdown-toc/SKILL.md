# Markdown TOC Skill

## 简介

该 Skill 帮助提取 Markdown 文件的目录层级结构，在进行笔记整理、文件修改或内容插入时，先了解文件的整体结构，提高操作效率和精准度。

## 作者

CS-Notes 项目

## 应用场景

### 笔记整理
当需要整理或修改 Markdown 笔记时，先提取目录层级，快速了解文件结构，避免盲目阅读。

### 内容插入
在寻找合适位置插入新内容时，通过目录树判断最适合的 section。

### 文件修改
在修改大型 Markdown 文件时，通过目录快速定位需要修改的章节。

```
帮我提取这个 Markdown 文件的目录
```


## 工作流程

1. 读取指定的 Markdown 文件
2. 解析文件中的标题层级（#、##、### 等）
3. 以树状结构展示目录
4. 根据目录结构提供操作建议

## 最佳实践

1. 在进行任何 Markdown 文件操作前，先提取目录
2. 对于大型文件，优先通过目录定位目标 section
3. 插入新内容时，参考目录结构选择最合适的位置

## 💡 沉淀经验

1. **永动机不舍昼夜**：把自己想象成永动机，不分昼夜推进todo，不要因为是"深夜"就停止推进todo！
2. **推进todo时主动给用户发消息**：在推进todo的时候，主动给用户发消息，告诉用户在推进什么todo，有什么进展！不要默默推进todo，让用户知道你在做什么！
3. **【重要】推进todo时要实质推进**：不要只是把任务的状态从 pending 改成 in-progress，并且加一句"永动机继续工作！"，要真正推进任务，做一些实质工作！
4. **【重要】更新 progress 字段时要包含实际进展**：不要只是用 "🚀 永动机继续工作！" 这样的空泛内容来更新 progress 字段，要包含实际进展的详细内容！
5. **Session 管理重要性**：Session 运行超过 24 小时后，建议切换新 session 以获得更好的性能和体验！
6. **MEMORY.md 定期更新**：定期更新 MEMORY.md 的内容，沉淀重要的经验和最佳实践！

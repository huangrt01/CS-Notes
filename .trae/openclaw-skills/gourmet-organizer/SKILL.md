# Gourmet Organizer

## 适用场景

当需要整理美食笔记、品鉴菜单时使用该技能。

## 功能特点

1. **品鉴菜单**：分析菜单内容，品鉴美食
2. **结合用户的美食品鉴经历**：结合 Gourmet.md 中用户的美食品鉴经历
3. **整理美食笔记**：将美食内容整理到 Gourmet.md 中
4. **可联网搜索**：可联网搜索美食相关信息
5. **隐性强调用户感受**：保留并融合强调用户的感受（不要指出是笔记记的，而是隐性地强调）

## 使用步骤

1. 准备美食相关内容（菜单、美食描述、美食照片等
2. 运行脚本 `python main.py "<美食内容>"
3. 脚本将整理美食笔记

## 示例

```bash
# 整理美食笔记
python main.py "今天吃了一道红烧鲍鱼，肉质厚弹牙，溏心黏糯，火候刚好"

# 从文件读取美食内容
python main.py --file menu.txt
```

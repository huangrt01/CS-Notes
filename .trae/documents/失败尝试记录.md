## 知乎文章抓取失败

**时间：** 2026-02-20

**尝试方法：**
1. web_fetch 工具直接请求知乎网页
2. Playwright 模拟浏览器抓取知乎文章

**跑的命令：**

### 1. web_fetch 工具直接请求
```bash
# 命令（通过 OpenClaw 工具调用）
web_fetch https://www.zhihu.com/question/1999487395494588876/answer/2000362433332660070?share_code=HGYNzglkEV8h&utm_psn=2000559952910099841
```

**错误信息：**
```
403 Forbidden
```

---

### 2. Playwright 模拟浏览器抓取
```bash
# 安装 Playwright
pip install playwright --break-system-packages

# 安装 Playwright 浏览器
playwright install chromium

# 运行 Playwright 抓取脚本
python3 Notes/snippets/playwright_simple.py "https://www.zhihu.com/question/1999487395494588876/answer/2000362433332660070?share_code=HGYNzglkEV8h&utm_psn=2000559952910099841" --save-html /tmp/zhihu-article.html --output /tmp/zhihu-result.json
```

**错误信息：**
```json
{
  "error": {
    "message": "您当前请求存在异常，暂时限制本次访问。如有疑问，您可以通过手机摇一摇或登录后私信知乎小管家反馈。cb62e8a486f488fa1e024a57b5118baf",
    "code": 40362
  }
}
```

---

### 3. ClawHub 安装 playwright-scraper-skill
```bash
# 搜索 skill
clawhub search playwright scraper

# 尝试安装 skill
clawhub install playwright-scraper-skill --dir .trae/openclaw-skills --force
```

**错误信息：**
```
Rate limit exceeded
```

---

## 总结

知乎的反爬机制非常强，即使使用 Playwright 模拟浏览器，还是返回了 403 错误。需要进一步探索其他方法，或者使用 ClawHub 中的 playwright-scraper-skill（但当前 rate limit exceeded）。

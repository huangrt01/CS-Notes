## 1. YouTube 频道视频定时抓取与摘要

### 场景
每天早上 8 点，抓取关注频道的最新视频，生成核心摘要。

### 提示语
```
每天早上 8 点，抓取下面 YouTube 频道的最新视频，生成一份包含核心摘要：@mreflow

针对每条新视频（过去 24-48 小时内发布）：
- 获取视频字幕
- 用 2-3 个要点总结核心内容
- 附上视频标题、频道名称及链接

特殊说明：
- 如果某个频道 ID 无法识别，请通过搜索找到正确的频道
- 请记住我的频道列表，方便我后续增删
```

## 2. YouTube 搜索新视频并去重处理

### 场景
每天上午 9 点搜索关于特定主题的新视频，避免重复处理。

### 提示语
```
每天上午 9:00，在 YouTube 上搜索关于"OpenClaw"的新视频。

排重机制：
- 维护一个名为 seen-videos.txt 的文件，记录已处理过的视频 ID
- 仅针对该文件中不存在的新视频获取字幕
- 处理完成后，将新视频 ID 存入 seen-videos.txt

单条视频处理要求：
- 获取视频字幕
- 生成 3 个要点的精简摘要
- 标注出任何与我工作相关的内容

先运行一次看看效果
```

## 3. 目标管理教练

### 场景
每天早上 8 点，根据长期目标主动策划当天可以完成的任务。

### 提示语 - 设置目标
```
这是我的个人目标，记下来并在后续所有任务中作为默认参考：

职业发展：
1. 小红书账号做到 10 万订阅
2. 建立可复现的内容方法论与系列栏目，持续增长与稳定爆款

产品发布：
1. 在 Q3 前发布我的产品
2. 以跑通核心闭环的 MVP 为交付标准，优先可用与迭代

个人成长：
1. 每月阅读 1 本设计相关书
2. 每本书输出结构化笔记，并落地到内容或产品中

决策原则：
1. 优先做能同时推动增长与产品的事
2. 优先做可复现可沉淀的事
```

### 提示语 - 每日任务策划
```
每天早上 8:00，请根据我的长期目标，主动策划 4-5 项你今天可以在我电脑上完成的任务。

（可选补充）根据今天我跟你对话信息开发一个能解决我问题的小应用原型（MVP）
```

## 4. 看板（Kanban Board）任务管理系统

### 场景
使用 Next.js 构建一个看板来展示任务状态。

### 提示语
```
请使用 Next.js 为我构建一个看板（Kanban Board），用于展示你正在处理的所有任务。

功能要求：
- 状态分列：看板需包含"待办 (To Do)"、"进行中 (In Progress)"和"已完成 (Done)"三个列表
- 实时更新：当你完成任务时，看板状态必须实时同步更新
```

## 5. 邮件订阅总结与筛选

### 场景
每天晚上 8 点总结过去 24 小时的订阅邮件。

### 提示语
```
我希望你每天晚上 8 点运行一次定时任务，读取过去 24 小时内的所有订阅邮件（Newsletter）。

用 Clawhub 搜索合适的 skills 完成这个任务。

执行细节：
1. 提炼精华：为我整理一份核心要点摘要，并附上原文链接以便深度阅读
2. 反馈互动：完成后询问我对本次内容筛选的评价
3. 自主进化：根据我的反馈更新你的记忆，不断优化筛选偏好，让未来的日报越来越精准
```

## 6. 第二大脑（Second Brain）系统

### 场景
搭建一个用于复盘笔记、对话记录和记忆碎片的系统。

### 提示语
```
我想搭建一个第二大脑系统 (Second Brain)，用于复盘我们所有的笔记、对话记录和记忆碎片。使用 Next.js 完成开发。

功能清单：
- 全量列表：一个支持搜索的列表，展示所有记忆点和历史对话
- 全局搜索 (Cmd+K)：支持通过快捷键唤起全局搜索，覆盖系统内所有内容
- 多维筛选：支持按日期和内容类型进行过滤
- 设计风格：界面需保持极致简约（Minimal UI）
```

## 7. 财报日历追踪与摘要生成

### 场景
每周日检索下周科技与 AI 公司的财报日历，为关注公司设置定时任务。

### 提示语
```
每周日 18:00，运行一次 Cron Job 执行以下操作：

1. 财报检索：搜索下周科技与 AI 公司的财报日历
2. 目标筛选：过滤出我关注的公司（如 NVDA, MSFT, GOOGL, META, AMZN, TSLA, AMD 等）
3. 人工确认：等待我确认具体要追踪哪几家公司

追踪逻辑（当我回复要追踪的公司后）：
- 精准调度：为每家公司的财报发布时间分别设置一个单次定时任务（One-shot Cron Job）
- 结果抓取：财报发布后，立即搜索业绩结果
- 格式化摘要：整理一份包含以下内容的报告：
  * 超预期/逊于预期（Beat/Miss）
  * 营收
  * 每股收益（EPS）
  * 核心指标
  * AI 相关亮点
  * 业绩指引（Guidance）
- 偏好记忆：记录我常追踪的公司，以便每周自动向我推荐
```

## 核心模式总结

1. **定时任务 + 记忆系统**：将长期信息获取任务加上定时和记忆功能
2. **去重机制**：维护文件记录已处理内容，避免浪费 Token
3. **目标驱动**：基于个人长期目标主动策划任务
4. **反馈循环**：根据用户反馈持续优化筛选偏好
5. **MVP 开发**：根据对话信息快速开发小应用原型
